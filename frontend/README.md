# 🎨 CodeZen - Frontend Client

This is the React client application for CodeZen, built with Vite, Tailwind CSS, and DaisyUI.

For the complete project setup, system architecture, and backend instructions, please refer to the main [Root README.md](../README.md).

##  Commands

### 1. Install dependencies
```bash
npm install
```

### 2. Start the development server
```bash
npm run dev
```

### 3. Build for production
```bash
npm run build
```
