# ⚙️ CodeZen - Backend Server

This is the Express.js API backend server for CodeZen.

For the complete project setup, system architecture, and frontend instructions, please refer to the main [Root README.md](../README.md).

## 🚀 Commands

### 1. Install dependencies
```bash
npm install
```

### 2. Seed database (creates Admin user and sample problems)
```bash
node seed.js
```

### 3. Start the server
```bash
node src/index.js
```
